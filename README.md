## :computer: Usage

### :rocket: Clone repository

```sh
git clone https://github.com/tocola/CppTemplate
```

### :warning: Requirement

- [CMake >= 3.17](https://cmake.org/download/)
- [C++17](https://en.cppreference.com/w/cpp/17)

## :checkered_flag: Quick-Start

```sh
# to build the program
make
./binary

# to build the tests
make tests
./unit_tests
```
